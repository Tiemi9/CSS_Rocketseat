# background

- define fundo para o elemento
- sua área de artuação é a caixa toda
- por padãro é trasparente

## Exemplos
- usar cores sólidas
- usar imagens
- controlar
    - a posição das imagens
    - se elas se repetem
    - o tamanho das janelas na caixa
- usar cor e iamgens juntas
- usar cor gradiente
